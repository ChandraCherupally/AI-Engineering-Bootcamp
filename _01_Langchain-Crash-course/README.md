## 📖 Overview

This repository contains my hands-on practice and learning exercises for **LangChain**, one of the most widely used frameworks for building LLM-powered applications, AI agents, Retrieval-Augmented Generation (RAG) systems, and multi-agent workflows.

The notebooks cover the fundamental building blocks required to develop production-ready Generative AI applications, including model integration, tool calling, structured outputs, message management, and middleware.
